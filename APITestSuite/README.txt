
Steps to run tests from command line:
1. Import project from Github
2. Open command prompt
3. Go to project path (until the project root directory APITestSuite) 
4. mvn test 
e.g. C:\Users\Suchita.Thorat\eclipse-workspace\APITestSuite>mvn clean test
-------------
OR Run using IDE:
Tests can be run in IDE with the help of runner file TestRunner.java
------------
Reports:
Cucumber test reports are generated at below path:
userdirectory\APITestSuite\target\cucumber-reports\index.html
------------
